# 🤖 MemoRemindMe Bot

Telegram-бот для нагадування важливих дат з підтримкою груп та природної мови українською.

## ✨ Можливості

- 🔤 **Природна мова**: пиши "завтра тест", "через 3 дні зустріч", "15 травня день народження"
- 📅 **Типи нагадувань**: разові, щомісячні, щорічні
- 👥 **Групи**: спільні нагадування для сім'ї, роботи, друзів (до 10 учасників)
- 📊 **Звіти**: щоденні, тижневі та місячні нагадування
- 🔐 **Доступ за паролем**: тільки авторизовані користувачі

## 🚀 Команди

| Команда | Опис |
|---------|------|
| `/start` | Почати роботу з ботом |
| `/menu` | Головне меню |
| `/add` | Додати нагадування |
| `/today` | Нагадування на сьогодні |
| `/week` | Нагадування на цей тиждень |
| `/month` | Нагадування на цей місяць |
| `/all` | Всі нагадування |
| `/help` або `/about` | Інформація про бота |
| `/cancel` | Скасувати поточну дію |

## 🛠 Технології

- Python 3.10+
- python-telegram-bot 20+
- SQLite
- python-dotenv
- dateparser
